브라우저 및 텔레그램 MiniApp 캐시 삭제 가이드

웹 브라우저 캐시 삭제

1. 크롬(Chrome) 브라우저에서 캐시 삭제하기
PC
  1. 크롬 브라우저 우측 상단의 점 세 개(⋮)를 클릭합니다.
  2. 「설정」을 선택합니다.
  3. 「개인정보 및 보안」을 클릭합니다.
  4. 「인터넷 사용 기록 삭제」를 선택합니다.
  5. 삭제할 기간을 「모든 기간」으로 설정합니다.
  6. 「쿠키 및 사이트 데이터」와 「캐시된 이미지 및 파일」을 체크합니다.
  7. 「데이터 삭제」 버튼을 클릭합니다.

빠른 방법: Ctrl + Shift + Delete 단축키를 누르면 인터넷 사용 기록 삭제 창이 바로 열립니다.

모바일
  1. 크롬 앱을 실행합니다.
  2. 우측 상단의 점 세 개(⋮)를 탭합니다.
  3. 「설정」을 탭합니다.
  4. 「개인정보 보호 및 보안」을 탭합니다.
  5. 「인터넷 사용 기록 삭제」를 탭합니다.
  6. 삭제할 기간을 「모든 기간」으로 설정합니다.
  7. 「쿠키 및 사이트 데이터」와 「캐시된 이미지 및 파일」을 체크합니다.
  8. 「데이터 삭제」 버튼을 탭합니다.
2. 사파리(Safari) 브라우저에서 캐시 삭제하기
PC (Mac)
  1. Safari 브라우저를 실행합니다.
  2. 상단 메뉴바에서 「Safari」를 클릭합니다.
  3. 「설정」을 선택합니다.
  4. 「개인정보 보호」 탭을 클릭합니다.
  5. 「웹 사이트 데이터 관리」를 클릭합니다.
  6. 「모든 웹 사이트 제거」를 클릭하거나 특정 사이트를 선택하여 삭제합니다.
  7. 「완료」를 클릭합니다.

또는 「Safari」 > 「기록 및 웹 사이트 데이터 지우기」를 선택해 빠르게 삭제할 수도 있습니다.

모바일 (iOS)
  1. iPhone 또는 iPad의 「설정」 앱을 실행합니다.
  2. 아래로 스크롤하여 「Safari」를 탭합니다.
  3. 「기록 및 웹 사이트 데이터 지우기」를 탭합니다.
  4. 확인 메시지가 나타나면 「기록 및 데이터 지우기」를 탭합니다.
3. 파이어폭스(Firefox) 브라우저에서 캐시 삭제하기
PC
  1. 우측 상단의 메뉴 버튼(≡)을 클릭합니다.
  2. 「설정」을 선택합니다.
  3. 「개인 정보 및 보안」을 클릭합니다.
  4. 「쿠키 및 사이트 데이터」 섹션에서 「데이터 삭제」를 클릭합니다.
  5. 「캐시된 웹 콘텐츠」와 「쿠키 및 사이트 데이터」를 체크합니다.
  6. 「삭제」 버튼을 클릭합니다.

빠른 방법: Ctrl + Shift + Delete 단축키를 누르면 바로 삭제 창이 열립니다.

모바일
  1. Firefox 앱을 실행합니다.
  2. 우측 하단의 메뉴 버튼(≡)을 탭합니다.
  3. 「설정」을 탭합니다.
  4. 「데이터 관리」를 탭합니다.
  5. 삭제하려는 데이터 유형(캐시, 쿠키 등)을 선택합니다.
  6. 「데이터 삭제」를 탭합니다.
4. 엣지(Edge) 브라우저에서 캐시 삭제하기
PC
  1. 우측 상단의 점 세 개(⋯)를 클릭합니다.
  2. 「설정」을 선택합니다.
  3. 「개인 정보, 검색 및 서비스」를 클릭합니다.
  4. 「검색 데이터 지우기」를 클릭합니다.
  5. 삭제할 기간을 「모든 기간」으로 설정합니다.
  6. 「쿠키 및 기타 사이트 데이터」와 「캐시된 이미지 및 파일」을 체크합니다.
  7. 「지금 지우기」 버튼을 클릭합니다.

빠른 방법: Ctrl + Shift + Delete 단축키를 누르면 바로 삭제 창이 열립니다.

모바일
  1. Edge 앱을 실행합니다.
  2. 하단의 점 세 개(⋯)를 탭합니다.
  3. 「설정」을 탭합니다.
  4. 「개인정보 및 보안」을 탭합니다.
  5. 「검색 데이터 지우기」를 탭합니다.
  6. 삭제할 기간과 데이터 유형을 선택합니다.
  7. 「데이터 지우기」를 탭합니다.
5. 삼성 인터넷 브라우저에서 캐시 삭제하기
모바일
  1. 삼성 인터넷 앱을 실행합니다.
  2. 우측 하단의 메뉴 버튼(≡)을 탭합니다.
  3. 「설정」을 탭합니다.
  4. 「개인정보 보호 및 보안」을 탭합니다.
  5. 「인터넷 사용 기록 삭제」를 탭합니다.
  6. 「캐시」와 「쿠키 및 사이트 데이터」를 체크합니다.
  7. 「삭제」 버튼을 탭합니다.

텔레그램 MiniApp 캐시 삭제

6. 텔레그램 PC에서 MiniApp 캐시 삭제하기
PC
  1. 텔레그램 데스크톱 앱을 실행합니다.
  2. 좌측 상단의 삼선 메뉴(≡)를 클릭합니다.
  3. 「설정」을 선택합니다.
  4. 「고급」을 클릭합니다.
  5. 「데이터 및 저장소」를 클릭합니다.
  6. 「캐시 지우기」를 클릭합니다.
  7. 「지우기」를 확인합니다.

이렇게 하면 모든 미니앱의 캐시가 함께 삭제됩니다. 특정 미니앱만 초기화하고 싶다면, 해당 미니앱을 다시 열어 새로고침(F5) 키를 누르거나, 미니앱 내의 설정에서 캐시 삭제 기능이 있는지 확인하세요.

7. 텔레그램 모바일에서 MiniApp 캐시 삭제하기
Android
  1. 텔레그램 앱을 실행합니다.
  2. 좌측 상단의 삼선 메뉴(≡)를 탭합니다.
  3. 「설정」을 탭합니다.
  4. 「데이터 및 저장소」를 탭합니다.
  5. 「저장소 사용량」을 탭합니다.
  6. 「캐시 지우기」를 탭합니다.
  7. 「지우기」를 확인합니다.
iOS
  1. 텔레그램 앱을 실행합니다.
  2. 하단의 「설정」 탭을 탭합니다.
  3. 「데이터 및 저장소」를 탭합니다.
  4. 「저장소 사용량」을 탭합니다.
  5. 「캐시 지우기」를 탭합니다.
  6. 「캐시 지우기」를 확인합니다.

텔레그램 앱을 완전히 종료하고 다시 실행하는 것도 MiniApp의 캐시를 초기화하는 데 도움이 될 수 있습니다. 또한 일부 MiniApp은 앱 내에서 자체적으로 캐시 삭제 옵션을 제공할 수 있으므로, MiniApp의 설정을 확인해보세요.

8. 특정 웹 게임/MiniApp에서 새로고침하기

캐시를 완전히 삭제하지 않고 웹 게임이나 MiniApp을 새로고침하는 방법:

PC
  • 대부분의 브라우저: F5 키 또는 Ctrl + R을 누릅니다.
  • 강제 새로고침(캐시 무시): Ctrl + F5 또는 Ctrl + Shift + R을 누릅니다.
  • 텔레그램 MiniApp: MiniApp 화면에서 F5 키를 누르거나, MiniApp을 닫고 다시 열어봅니다.
모바일
  • 대부분의 브라우저: 화면을 아래로 끌어당겨 새로고침합니다.
  • 텔레그램 MiniApp: MiniApp을 닫고 다시 열거나, 텔레그램 앱을 완전히 종료한 후 다시 실행합니다.
9. 캐시 삭제 후에도 문제가 해결되지 않을 경우

캐시를 삭제해도 문제가 지속될 경우 다음 방법을 시도해보세요:

  • 브라우저나 텔레그램 앱을 최신 버전으로 업데이트합니다.
  • 다른 브라우저로 접속해 봅니다.
  • 모바일 데이터와 Wi-Fi를 전환해봅니다.
  • 디바이스를 재부팅합니다.
  • 웹 게임이나 MiniApp 개발자에게 문의합니다.

일부 게임이나 앱의 경우, 로그인 정보가 서버에 저장되어 있어 캐시 삭제만으로는 초기화되지 않을 수 있습니다. 이 경우 계정 로그아웃 후 다시 로그인해보세요.

10. 정기적인 캐시 관리가 필요한 이유

정기적인 캐시 관리가 중요한 이유:

  • 웹 게임 성능 향상: 오래된 캐시 파일은 게임 로딩 속도를 저하시킬 수 있습니다.
  • 버그 해결: 업데이트된 게임이나 앱의 새 버전이 제대로 로드되지 않을 수 있습니다.
  • 저장 공간 확보: 캐시는 시간이 지남에 따라 많은 저장 공간을 차지할 수 있습니다.
  • 데이터 보호: 캐시에는 개인 정보가 포함될 수 있어 주기적인 삭제가 보안에 도움이 됩니다.

대부분의 경우 1-3개월마다 한 번씩 캐시를 정리하는 것이 좋습니다. 웹 게임이나 MiniApp에 문제가 발생했을 때는 언제든지 캐시를 삭제해보세요.
